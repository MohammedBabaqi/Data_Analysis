
# Data Analysis Notebook

A Python-based data analysis project that cleans, processes, and visualizes order data.

## Features

- **Data Import & Cleaning**: Load CSV data and handle missing values
- **Data Transformation**: Fix columns, standardize formats, calculate totals
- **Aggregation**: Group and summarize data by country, category, and item
- **Visualization**: Generate pie charts and bar charts for insights

## Files

- `Data_Analysis_notebook.ipynb` - Main analysis notebook
- `Orders.csv` - Source data file

## Requirements

```
pandas
matplotlib
```

## Usage

Run the notebook cells in order to:

1. Import and explore the data
2. Clean and standardize values
3. Generate aggregated reports
4. Create visualizations
